About

This is a real-time NBA player data visualization project. You can find all player's data by the team, such as Height, Weight, Jersey Number, Career Points, Career Assists, and Career Rebounds.

How to Search

You have to type teams first name second name or full name like(Sacramento, Sacramento Kings, Kings)
